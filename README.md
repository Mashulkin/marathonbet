To run this application


**EPL**

export SIMPLE_SETTINGS=settings.epl

python teamsodds.py


**Championship**

export SIMPLE_SETTINGS=settings.championship

python teamsodds.py


**Laliga**

export SIMPLE_SETTINGS=settings.laliga

python teamsodds.py

**Bundesliga**
